# api-jwt-mongo

Proyecto ejemplo: Express + JWT + MongoDB + Jest

Setup rápido:

1. Copiar variables de entorno: `cp .env.example .env`
2. Instalar dependencias: `npm install`
3. Ejecutar en desarrollo: `npm run dev`

Tests:

```
npm test
```

Despliegue:

- Para Vercel configure `VERCEL_TOKEN` y secretos de su proyecto y añada un workflow que despliegue o use la integración Vercel.
